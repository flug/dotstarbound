HOVERBIKES

=== HOVERBIKES 0.1 ===

Мне всегда хотелось полетать на ховербайке. И я решил сделать для себя и остальных, кого не покидало это желание данный маленький мод.

Содержит два ховербайка для двух рас:
- Ховербайк людей, имеет классические квадратные формы, желтую раскраску и пониженый расход топлива.
- Ховербайк хайлотлов, вдохновленный креведками дизайн и быстрая скорость - вот отличительные черты данной машины.

В будущем возможно добавятся ховербайки остальных рас.

== Установка и использование ==

Просто разархивируйте "Hoverbikes_0.1.zip" в папку 'Starbound/mods'.

На существующих персонажах для появления рецепта нужно еще раз использовать Cerulium Tech Upgrade.

Чипы технологий можно создать из чипа технологии Human Mech и 50 слитков велириума на роботизированном столе.

Для продвинутых пользователей - возможно выдать себе следующие два айтема, которые хотят просто потестировать:
- human_hoverbike-chip
- hylotl_hoverbike-chip

== COMPATABILITY ==

Данных нет. Если у вас есть какие-то данные о несовместимости данного мода - пожалуйста сообщите, я учту в будущих релизах.
